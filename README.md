### Shipped Projects

#### 🔐 [Calculator Lock](https://github.com/afaan13/CalculaterAppLock--Releases)
Privacy suite disguised as a calculator. Locks apps, encrypts photos/videos with AES-256, captures intruder selfies on wrong PIN attempts.

**Tech:** Kotlin, Jetpack Compose, Room DB, Accessibility Service for app blocking, encrypted SharedPreferences

**Key Challenge:** Building a reliable app-blocking mechanism using AccessibilityService while handling Android's aggressive background process killing.

---

#### 📋 [Attendance Tracker](https://github.com/afaan13/attendance-tracker-releases)
Offline-first attendance management for teachers. Mark attendance, sync across devices via Firebase, generate PDF reports, share school workspace with colleagues using invite codes.

**Tech:** Kotlin, Jetpack Compose, Firebase (Auth + Firestore), Room DB for offline storage, PDF generation

**Key Challenge:** Real-time multi-user sync without conflicts when multiple teachers mark attendance simultaneously.

---

#### ⏰ [WakeForce](https://github.com/afaan13/Force-Wake-clock-releases)
Mission-based alarm app that requires completing tasks (math problems, QR scanning, step counting, voice phrases) to dismiss. Built to solve the "snooze loop" problem for heavy sleepers.

**Tech:** Flutter, Kotlin (native layer for system-level alarm management), SQLite, full-screen intents for lockscreen overlay

**Key Challenge:** Ensuring alarm fires reliably across Android power management modes (Doze, App Standby) using WorkManager + AlarmManager hybrid approach.
